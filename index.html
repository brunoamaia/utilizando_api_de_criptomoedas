Uso de API para consultar propriedades de Criptomoedas

Página criada para requisitar dados sobre criptomoedas para a API da CoinMarketCap.

Quando a página é iniciada, os dados são requisitados e posteriormente utilizados para a construçãoda tabela. Após a consulta, os dados são salvos na seção do navegador. Dessa forma, para as próximas ações ou carragamento da página, não é necessário utilizar a API (diminuindo o tráfego na rede e não "estourando" o limite de acesso a API).

Devido a limitações da API utilizada, é necessário cumprir alguns pré-requisitos:

  • Utilizar o navegador Google Chrome
